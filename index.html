Getting Started

Setup Guide

SharePoint Setup (for shared access)

Follow these 4 steps so all team members share the same data.

1
Create list SS_TeamRoster
Columns: Title (name), Email, Role, Color, Active (Yes/No), ExtID (Number)
2
Create list SS_PrioritySubmissions
Columns: Title (week key), MemberID, MemberName, Priorities, Notes, SubmittedAt, OnTime
3
Upload this HTML
Upload to your SharePoint Site Assets library. Open the file URL in browser.
4
Set your site path
In the Team tab, enter your SharePoint site path (e.g. /sites/StellarSeeds). The page auto-connects.
Power Automate emails: Create scheduled flows for Monday reminders (8AM) and Tuesday follow-ups (8AM) using the SS_PrioritySubmissions and SS_TeamRoster lists as data sources. See the Setup tab in previous conversations for detailed flow instructions.
Without SharePoint: The tracker works in local mode without any setup. Each person's data stays in their own browser. Use this for testing before going live.
