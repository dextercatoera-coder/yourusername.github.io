Getting Started

Setup Guide

Connect to Google Sheets (5 minutes)

This lets all team members see the same data in real-time.

1
Create a Google Sheet at sheets.google.com. Name it "Stellar Seeds Priority Tracker".
2
Create 2 tabs:
Tab 1: Submissions — Headers: WeekKey | MemberID | Name | Priorities | Notes | SubmittedAt | OnTime
Tab 2: Members — Headers: ID | Name | Role | Email | Color | Active
3
Add team members in the Members tab (rows 2+). Set Active to true.
4
Click Extensions → Apps Script. Delete all code. Paste the script from the box below. Save it.
5
Deploy → New Deployment → Gear icon → Web app → Execute as: Me → Who has access: Anyone → Deploy → Copy the URL.
6
Paste the URL in the Team tab → Google Apps Script URL field. The connection bar turns green.
7
Share this page URL with your team. Everyone enters the same Apps Script URL in the Team tab.
Apps Script code: Copy the full code from this setup, or ask your manager for the code. It handles reading/writing to the Google Sheet automatically.
Without Google Sheets: The tracker still works in local mode. Each person's data stays in their browser. Use this for testing before connecting.
